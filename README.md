# 🚀 AI File Processing SaaS
<h4> Under Development </h4>

An AI-powered web application that allows users to process files (PDFs & Images) using natural language commands.

Instead of manually selecting tools, users can simply type:
> "Compress this PDF to 2MB"  
> "Resize this image to 800x600"

The system understands the intent and performs the task automatically.

---

## 🧠 Features

- 📂 Upload PDF or Image files
- 💬 Natural language prompt input
- ⚙️ Automatic file processing based on user intent
- 📥 Download processed files
- 🔄 Real-time backend processing

### APIs & Tools
- ILovePDF API (PDF Compression)
- ConvertAPI (File Conversion)
- Pillow (Image Processing)
